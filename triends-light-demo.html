Triends Light Village Demo

A trimmed standalone slice focused only on the Light Village puzzle logic.

This version isolates the light mechanic from the larger TRIENDS project so visitors can quickly try the core idea inside the portfolio site: shoot the lamp, send light through the wire, and clear the glowing flowers.

Back to TRIENDS More Projects

Desktop is recommended for this demo. Use a keyboard to move and shoot.

Objective

Move close to the lamp and fire light into it.

Controls

  • Move: Arrow keys or WASD
  • Shoot light: Space
  • Restart: R or the restart button

What is included

Lamp activation Wire propagation Flower reaction Map crop

Why this version exists

The original TRIENDS project is a larger cooperative adventure. For the portfolio, I separated one mechanic-driven section into a lighter standalone page so the key interaction can be experienced directly instead of only being shown in video form.
